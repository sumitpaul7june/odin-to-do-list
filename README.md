# To-Do List App

A fully functional to-do list application built with **vanilla JavaScript** and **Webpack**, featuring project-based task organization and persistent storage using `localStorage`.
This project was created as part of **The Odin Project** curriculum to practice modular JavaScript, state management, and Webpack workflows.
